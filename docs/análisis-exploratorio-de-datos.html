Análisis Exploratorio de Datos: Tasa de Finalización de Educación Primaria Femenina

Chapter 2 Análisis exploratorio de datos

2.1 Análisis univariado

En el período 2000–2022 contamos con datos de 192 países, aunque la cobertura es heterogénea: solo 30 de ellos (15,6%) reportan la serie completa de 23 años, mientras que el resto presenta huecos en distintos momentos. En términos de extensión temporal por país, el mínimo es de 1 año, el p25 es 11, la mediana es 17 y el p75 es 21. Estos resultados confirman que trabajamos con un panel desbalanceado; por ello, los análisis temporales y las comparaciones entre países deben considerar esta asimetría para evitar sesgos en la interpretación.

Es importante señalar que la tasa de finalización primaria puede superar el 100%. Esto ocurre porque el numerador contabiliza a todos los estudiantes que egresan del último grado de primaria —independientemente de su edad—, mientras que el denominador solo incluye la población oficialmente asignada a ese nivel. La presencia de estudiantes en sobredad, subedad o con historial de repetición puede inflar el numerador. Para mejorar la claridad visual, los gráficos trabajan con una versión acotada de la variable en el rango 0–100, conservando los valores originales para efectos de trazabilidad.

La distribución de la tasa de finalización primaria es marcadamente asimétrica hacia la izquierda: la mayoría de los países reportan valores altos, con una mediana cercana al 100%, pero existe una cola importante hacia valores bajos que evidencia rezagos persistentes en un grupo significativo de naciones. Esta asimetría se refleja en que la media es inferior a la mediana, indicando que unos pocos países con tasas muy bajas jalan el promedio hacia abajo. El boxplot muestra además valores atípicos en la parte inferior del rango, correspondientes a países con serias dificultades para retener a las niñas en el sistema escolar.

2.2 Distribución geográfica

Un mapa coropleta permite identificar patrones espaciales que los gráficos estadísticos no revelan: qué regiones del mundo concentran los mayores rezagos, si los países vecinos tienden a comportarse de manera similar y dónde se ubican los extremos del indicador. Para construirlo, calculamos la tasa de finalización promedio por país a lo largo de todo el período disponible (2000–2022).

df_mapa <- df_2 %>%
  group_by(pais) %>%
  summarise(tasa_promedio = mean(tasa_fin_cap, na.rm = TRUE), .groups = "drop")

world <- ne_countries(scale = "medium", returnclass = "sf")

world_data <- world %>%
  left_join(df_mapa, by = c("name_long" = "pais"))

ggplot(world_data) +
  geom_sf(aes(fill = tasa_promedio), color = "white", linewidth = 0.1) +
  scale_fill_gradientn(
    colours  = c(rojo_out, "#F5A623", "#F8E71C", azul_main, azul_fuerte),
    na.value = "#CCCCCC",
    name     = "Tasa promedio (%)",
    limits   = c(0, 100),
    breaks   = c(0, 25, 50, 75, 100)
  ) +
  labs(
    title    = "Tasa de finalización primaria femenina — Promedio 2000–2022",
    subtitle = "Gris: países sin datos disponibles en WDI",
    caption  = "Fuente: World Development Indicators, Banco Mundial"
  ) +
  theme_void(base_size = 12) +
  theme(
    plot.title       = element_text(face = "bold", hjust = 0.5, size = 13),
    plot.subtitle    = element_text(hjust = 0.5, color = gris_linea),
    plot.caption     = element_text(hjust = 1, color = gris_linea, size = 9),
    legend.position  = "bottom",
    legend.key.width = unit(2, "cm")
  )

El mapa revela patrones geográficos marcados. África Subsahariana concentra los valores más bajos, con varios países en el rango del 30–60%, lo que refleja dificultades estructurales en acceso y retención escolar. En contraste, América Latina, Europa, el norte de África y buena parte de Asia presentan tasas promedio superiores al 80%. El sur y sureste asiático muestran heterogeneidad importante: mientras algunos países han avanzado considerablemente, otros mantienen rezagos notables. Los países en gris no cuentan con datos suficientes en el WDI durante el período analizado.

2.3 Análisis bivariado

Para el análisis bivariado trabajamos con la versión acotada de la variable objetivo (tasa_fin_cap ∈ [0,100]), con el fin de asegurar visualizaciones comparables y evitar que pocos valores extremos distorsionen las escalas. Dado el panel desbalanceado, se emplean medidas robustas a asimetrías como la mediana y la banda intercuartílica.

2.3.1 tasa_fin_cap según país

Los siguientes gráficos muestran la distribución de tasa_fin_cap para los 8 países con mayor cobertura temporal, desagregada por quinquenio. El gráfico de lollipop presenta la mediana (punto) y el rango intercuartílico (segmento) para cada país y período.

df_base <- df_2 %>%
  filter(!is.na(pais), !is.na(anio), !is.na(tasa_fin_cap),
         anio >= 2000, anio <= 2022) %>%
  mutate(
    quinquenio = case_when(
      anio <= 2004 ~ "2000-2004",
      anio <= 2009 ~ "2005-2009",
      anio <= 2014 ~ "2010-2014",
      anio <= 2019 ~ "2015-2019",
      TRUE         ~ "2020-2022"
    ),
    quinquenio = factor(quinquenio,
      levels = c("2000-2004","2005-2009","2010-2014","2015-2019","2020-2022"))
  )

cov_por_pais <- df_base %>%
  distinct(pais, anio) %>%
  count(pais, name = "n_anios") %>%
  arrange(desc(n_anios))

paises_top <- cov_por_pais %>% slice(1:8) %>% pull(pais)

df_top <- df_base %>%
  filter(pais %in% paises_top) %>%
  mutate(pais_lab = str_wrap(pais, width = 18))

res_q <- df_top %>%
  group_by(quinquenio, pais_lab) %>%
  summarise(
    med = median(tasa_fin_cap, na.rm = TRUE),
    p25 = quantile(tasa_fin_cap, 0.25, na.rm = TRUE),
    p75 = quantile(tasa_fin_cap, 0.75, na.rm = TRUE),
    .groups = "drop"
  ) %>%
  group_by(quinquenio) %>%
  mutate(pais_ord = fct_reorder(pais_lab, med, .desc = TRUE)) %>%
  ungroup()

ggplot(res_q, aes(x = pais_ord)) +
  geom_hline(yintercept = 100, linetype = "dashed",
             color = gris_linea, linewidth = 0.5) +
  geom_segment(aes(xend = pais_ord, y = p25, yend = p75),
               color = gris_linea, linewidth = 2, alpha = 0.8) +
  geom_point(aes(y = med),
             color = azul_fuerte, fill = azul_main,
             size = 3.2, shape = 21, stroke = 0.6) +
  coord_flip() +
  facet_wrap(~ quinquenio, ncol = 3) +
  labs(
    title    = "tasa_fin_cap por país (Top 8) — Mediana e IQR por quinquenio",
    subtitle = "Escala 80–100. Línea discontinua: techo (100).",
    x = NULL, y = "tasa_fin_cap (%)"
  ) +
  scale_y_continuous(limits = c(80, 100), breaks = seq(80, 100, 5),
                     expand = expansion(mult = c(0.01, 0.03))) +
  theme_minimal(base_size = 12) +
  theme(
    plot.title       = element_text(face = "bold"),
    axis.text.y      = element_text(size = 9, lineheight = 0.9),
    panel.grid.minor = element_blank(),
    strip.text       = element_text(size = 10)
  )

En los 8 países con mayor cobertura temporal se observa una convergencia hacia valores altos de la tasa de finalización, con medianas cercanas o iguales a 100 en la mayoría de los quinquenios. Brunei Darussalam y Argentina muestran una compresión fuerte contra el techo desde los primeros períodos, lo que limita la comparabilidad fina entre ellos. Por su parte, Azerbaiyán y Cabo Verde exhiben trayectorias de mejora gradual con reducción de la dispersión a lo largo del tiempo, mientras que Cuba se mantiene alta y estable sin tocar el techo, lo que permite distinguir variaciones reales entre años.

La mediana anual muestra una tendencia claramente ascendente a lo largo de 2000–2022, evidenciando un progreso sostenido en la finalización primaria femenina a nivel global. La banda intercuartílica se desplaza hacia arriba y se estrecha en los años finales, lo que sugiere una convergencia parcial entre países. La mediana supera sistemáticamente a la media, confirmando la asimetría de la distribución: unos pocos países con tasas bajas presionan el promedio hacia abajo, aunque esta brecha se reduce progresivamente hacia el final del período.

2.3.3 tasa_fin_cap según país — ritmo de mejora anual

Los países no mejoran al mismo ritmo. Para cuantificar el avance de cada nación, estimamos la pendiente de una regresión lineal simple de tasa_fin_cap sobre anio para cada país. Los resultados se presentan gráficamente y en un mapa que ubica espacialmente dónde se produjeron las mejoras más significativas.

# Pendiente por país antes de tocar el techo
slopes <- df_num %>%
  group_by(pais) %>%
  arrange(anio, .by_group = TRUE) %>%
  mutate(
    capped        = tasa_fin_cap >= 100 - 1e-9,
    first_cap_year = ifelse(any(capped), min(anio[capped]), Inf)
  ) %>%
  filter(anio < first_cap_year | is.infinite(first_cap_year)) %>%
  mutate(n_pts = n()) %>%
  filter(n_pts >= 4) %>%
  do({
    fit <- lm(tasa_fin_cap ~ anio, data = .)
    tibble(pendiente = unname(coef(fit)[["anio"]]), n_usados = nrow(.))
  }) %>%
  ungroup() %>%
  arrange(desc(pendiente))

# Gráfico de barras — Top 10 países con mayor mejora
top_mejora <- slopes %>% slice(1:10)

ggplot(top_mejora, aes(x = reorder(pais, pendiente), y = pendiente)) +
  geom_col(fill = azul_main, color = azul_fuerte, width = 0.7) +
  geom_text(aes(label = round(pendiente, 2)),
            hjust = -0.1, size = 3.2, color = gris_linea) +
  coord_flip(clip = "off") +
  labs(
    title    = "Top 10 países con mayor mejora anual en tasa_fin_cap",
    subtitle = "Pendiente estimada (pp/año) — solo países con ≥ 4 años de datos pre-techo",
    x = NULL, y = "Puntos porcentuales por año"
  ) +
  scale_y_continuous(expand = expansion(mult = c(0, 0.15))) +
  theme_minimal(base_size = 12) +
  theme(
    plot.title       = element_text(face = "bold"),
    panel.grid.minor = element_blank(),
    plot.margin      = margin(t = 5, r = 25, b = 5, l = 5)
  )

# Mapa de pendientes
world_slopes <- ne_countries(scale = "medium", returnclass = "sf") %>%
  left_join(slopes %>% rename(pendiente_anual = pendiente),
            by = c("name_long" = "pais"))

ggplot(world_slopes) +
  geom_sf(aes(fill = pendiente_anual), color = "white", linewidth = 0.1) +
  scale_fill_gradientn(
    colours  = c(rojo_out, "#F5F5F5", azul_fuerte),
    na.value = "#CCCCCC",
    name     = "pp/año",
    limits   = c(-3, 8)
  ) +
  labs(
    title    = "Ritmo de mejora anual de la tasa de finalización primaria femenina",
    subtitle = "Azul: mayor mejora. Rojo: deterioro. Gris: sin datos suficientes.",
    caption  = "Fuente: World Development Indicators, Banco Mundial"
  ) +
  theme_void(base_size = 12) +
  theme(
    plot.title       = element_text(face = "bold", hjust = 0.5, size = 13),
    plot.subtitle    = element_text(hjust = 0.5, color = gris_linea),
    plot.caption     = element_text(hjust = 1, color = gris_linea, size = 9),
    legend.position  = "bottom",
    legend.key.width = unit(2, "cm")
  )

Los países con mayor ritmo de mejora son, en su mayoría, naciones que partían de niveles bajos de finalización primaria, lo que refleja un efecto de “catch-up”: cuando hay más margen de mejora, los avances tienden a ser más rápidos. El mapa de pendientes complementa esta lectura con una dimensión espacial: el azul intenso se concentra en África Subsahariana y partes del sur de Asia, precisamente las regiones que partían de niveles más bajos y donde el margen de avance era mayor. Los países en tonos neutros o rojizos reflejan estancamiento o ligero deterioro, a menudo asociado a contextos de inestabilidad política o crisis económica. Esta distribución geográfica refuerza la hipótesis del efecto “catch-up” y sugiere que las políticas de expansión educativa tuvieron un impacto especialmente pronunciado en los contextos con mayor rezago inicial.
